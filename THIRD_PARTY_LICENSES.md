# Third-Party Dependencies

This project uses the following third-party libraries:

## Chart.js

- **Version**: 4.4.0
- **License**: MIT License
- **Source**: https://github.com/chartjs/Chart.js
- **Usage**: Used in status page generation for visualizing response time trends and uptime metrics
- **Delivery**: Loaded via CDN (jsDelivr) - not bundled with this project
- **License URL**: https://github.com/chartjs/Chart.js/blob/master/LICENSE.md

## PHP Extensions Required

The following PHP extensions are required but are part of PHP core/standard:
- **cURL**: For HTTP requests (bundled with PHP)
- **JSON**: For data storage (bundled with PHP)

These are standard PHP extensions and do not add licensing requirements.
